# From-Scratch-Transformers
Resolução de Atividade Facultativa

Este codigo precisa de um programa com Python 3.0 ou uma versão mais recente, precisa-se da biblioteca do Numpy para as operações funcionarem devidamente.

Partes do codigo foram complementadas pela IA Claude, Revisadas por Guilherme De Assis. Mais especificamente em Sintaxes da biblioteca Numpy e auxilio no entendimento de funções logicas aplicadas no codigo.
